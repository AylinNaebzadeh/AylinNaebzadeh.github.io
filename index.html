Hi, I’m Aylin Naebzadeh—a 2024 Computer Engineering graduate from the University of Science and Technology in Tehran. I began as a backend developer at the Advanced Big Data Analysis Laboratory (ABDAL), working on spatio-temporal data, API improvements, log management, and data migration from MySQL to Elasticsearch. Inspired by advances in AI and LLMs, I moved into NLP research, contributing to a Persian psychological health chatbot. I now work as an AI developer at Dadmatech, enhancing Dadmatools, an open-source toolkit for the Persian language.
